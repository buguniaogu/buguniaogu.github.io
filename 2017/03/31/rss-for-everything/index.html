rss_for_everything

Posted on 2017-03-31

GitHub · Build software better, together. · by xzonepiece

在一个RSS阅读器中阅读所有信息源提供的内容，包括微信公众号、简书、知乎、博客、图虫、Lofter、Instagram。。。

1.订阅源

1.1. 订阅微信公众号

其实，订阅微信公众号是非常麻烦的，在这里推荐使用微广场，它提供微信公众号的全文RSS，免费用户可以订阅10个公众号，而且基本上热门的公众号都能找到。

1.2. 订阅简书

至于简书，推荐一个网站Jianshu RSS，使用方法非常简单，直接将你想订阅的简书专题或作者主页的网站粘贴到输入框中，点击GO就可以生成对应的RSS源。

1.3. 订阅知乎专栏

订阅知乎专栏可使用这个网站提供的服务，但是，因为知乎上的图片好像有图片防盗链设置，为了让图片能够正确显示，浏览器上需要安装一些插件，比如：RefControl，或My Webrequest，使用方法参见插件自身的教程。

1.4. 订阅博客

大部分博客都是提供RSS订阅的，图虫、Lofter这类图片类轻博客也是支持的，但是如果网站不提供怎么办，在这里你可以使用feed43，我在之前的文章中也提到过，使用教程可以参见小众软件，操作也比较简单。 （feed43似乎付费才能好用，操作稍困难）转到1.5

1.5. 订阅任意网站

如果你想将任意网站变成RSS的话，你可能需要自己去写代码啦，不过也有一些软件可以做到，在这里强烈推荐一款开源应用—Huginn，需要自己将代码部署到云端，喜欢折腾的人可以去试试，官方教程很详细。它的功能非常强大，不仅仅是能将任意网站变成RSS，还可以实现很多自定义功能，有点类似IFTTT。想尝鲜的人可以试试有人搭建好的Huginn网站，邀请码是try-huginn，烧制RSS的教程参见这篇文章—Huginn: 烧录 RSS 的神器。前面我提到微口网只提供摘要RSS，这样的话，你就可以使用Huginn将摘要RSS变成全文RSS，你也可以将传送门或爱微帮上的公众号变成RSS。

1.6. 订阅微博

如果你需要将微博上的一些账号变成RSS的话，可以试试微博档案。 （建议RSS Factory或Weibo2RSS ，最好转到Telegram，而非RSS）

1.7. 订阅Instagram、Twitter、Facebook等国外社交网站 (这类最好通过Ifttt转到Telegram，而非RSS)

在国内，我们主要使用的社交平台是微信、QQ和微博，但是，我也上Instagram、Twitter或Facebook，不过一般只是为了浏览一下我关注的账号有无更新，很少去评论，最多也就点个赞。但是，为了看这些内容，我们需要翻墙、登录好几个网站，这非常地麻烦，所以如果你的目的和我一样的话，不妨将这些网站的信息源制作成RSS吧，方法很简单，使用rssbridge提供的服务即可，它支持Instagram、Twitter、Facebook、Pinterest，甚至还可以将谷歌搜索变成RSS。

  1. RSS相关工具

下面推荐一些RSS相关的工具：

  • Huginn：神器，不仅仅是制作RSS（任意网站、邮件等等）

  • feed43：可以为静态网站制作RSS

  • feedmyinbox：可以将RSS发送到指定邮箱

  • IFTTT：可以将RSS与其他应用连接起来

  • Reabble：通过该服务可以在kindle上阅读RSS

  • feedburner：烧制RSS，可以用来托管自己制作的RSS

  • Instant RSS Search：RSS搜索引擎

  • feedvalidator：检验RSS源是否有效

  • fivefilters：可以将摘要RSS变成全文RSS

  • RSS Factory：一个用来生成微博RSS的Web APP

  • Weibo2RSS：生成微博RSS的开源工具

  • 即刻：基于RSS理念的一款APP

  • RSS Subscription Extension：chrome插件，为浏览器添加一键订阅RSS功能，自动发现网页上的RSS源
